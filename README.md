# godville-duel-entity
Тактический «радар» для арен (и тренировок) в Годвилле. Скрипт анализирует математику боя, ведет учет статистики (отслеживает циклы кривоты, гласы) и рассчитывает боевую эффективность в моменте (Удачу).
## 🚀 Основные возможности

*   **Счетчик Удачи (Efficiency)**: Рассчитывает ваш реальный КПД. Помогает понять, на чьей стороне сейчас «ветер рандома».
*   **Словарь Кривоты**: Автоматически распознает осечки влияний (X) и чистые попадания (0). Учитывает гендерные особенности текста (поймал/поймала).
*   **Вероятность кривоты**: Предсказывает шанс в текущем цикле на основе истории ходов (цикл из 6(4) раундов).
*   **Боевой LOG**: Компактная боковая панель со всей историей боя в символах.

## 📝 Расшифровка интерфейса и LOG

Скрипт переводит события боя в простые символы для быстрого анализа:

*   **0 — Влияние**: Успешное использование праны (сделать «хорошо» или «плохо»).
*   **X — Кривая**: Влияние не сработало, оказало эффект на обоих или повлияло негативно (отлечило врага или ударило по вам).
*   **г — Глас**: Вы отправили текстовое указание герою (бей, лечись, молись).
*   **э — Эффект гласа**: Глас сработал успешно (герой внял тексту и произвел нужное действие).
*   **--- — Ожидание**: Свободные слоты(следующие влияния) в текущем цикле из 6 раундов.

## 📊 Математика Удачи

Мы считаем боевую эффективность, деля **Профит** (урон врагу + ваше лечение) на **Вес хода**:

*   **Белый удар (база)**: Вес = `1.0`
*   **Глас (г)**: +`1.1` к весу
*   **Влияние (0/X)**: +`2.0` к весу
*   *Итого: Глас + Влияние за один ход дают суммарный вес `3.1`.*

**Цвета Luck:**
*   🟢 **Зеленый**: Ваш КПД выше, чем у противника. Вы эффективнее распоряжаетесь ресурсами.
*   🔴 **Красный**: Противнику везет больше. Его действия приносят больше результата на единицу усилий.

## 🕹 Управление

1.  **SCAN**: Принудительно пересканировать страницу (если данные не обновились автоматически).
2.  **LOG**: Открыть/закрыть боковую панель с подробной историей ходов.
3.  **RESET**: Полный сброс данных (нужно при переходе к новому бою без перезагрузки страницы).

## 🛠 Установка

1. Установите расширение [Tampermonkey](https://tampermonkey.net) (или аналоги).
2. Создайте новый скрипт и вставьте код из файла `tracker.user.js`.
3. Либо откройте `tracker.user.js` в этом репозитории, нажмите кнопку **Raw** и подтвердите установку.
